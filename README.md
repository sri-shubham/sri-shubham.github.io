## Local Development

To run this site locally, you must first install [Hugo](https://gohugo.io/installation/) (the extended version is recommended).

```bash
# Clone the repository (if you haven't already)
git clone https://github.com/sri-shubham/sri-shubham.github.io.git
cd sri-shubham.github.io

# Start the Hugo development server
hugo server -D
```

Then open `http://localhost:1313` in your browser to view the site.
